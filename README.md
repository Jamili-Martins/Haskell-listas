# Listas de exercícios - Haskell 🚀

Este repositório contém as minhas soluções para as listas de exercícios de Haskell. O objetivo é registrar o progresso e o aprendizado da linguagem de forma gradual, explorando desde conceitos básicos até lógica mais avançada.

### Listas adicionadas até o momento:

1. **[Lista 1]** Introdução, listas e definição de funções (21 questões)
2. **[Lista 2]** Compreensão de listas (8 questões)
3. **[Lista 3]** Recursão (8 questões)

---

### 📂 Estrutura e Dicas

* **Onde encontrar:** Todos os exercícios estão disponíveis nos arquivos `.hs` na raiz deste repositório.
* **Comentários:** Os exercícios possuem o enunciado comentado no próprio código, facilitando a leitura e o entendimento da lógica aplicada.

---
**Dica:** Para rodar os códigos, você pode utilizar o **GHCi** diretamente no terminal para testar as funções de cada lista.
